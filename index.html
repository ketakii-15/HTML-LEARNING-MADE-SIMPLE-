Introduction

HTML stands for HyperText Markup Language. It is the standard markup language used for creating web pages.

HTML stands for Hypertext Markup Language, and it is the most widely used language to write Web Pages.
 Hypertext refers to the way in which Web pages (HTML documents) are linked together. Thus, the link available on a webpage is called
 As its name suggests, HTML is a Markup Language which means you use HTML to simply "mark-up" a text document with tags that tell a Web browser how to structure it to display. Originally, HTML was developed with the intent of defining the structure of documents like headings, paragraphs, lists, and so forth to facilitate the sharing of scientific information between researchers. Now, HTML is being widely used to format web pages with the help of different tags available in HTML language.

HTML Elements

HTML elements are the building blocks of web pages. They can be used to create headings, paragraphs, lists, images, and much more.

HTML TAGS

HTML (Hypertext Markup Language) tags are used to create web pages and define the structure, content, and layout of the page. Here are some commonly used HTML tags:


1.html: This tag indicates the beginning of an HTML document.

2.head: This tag is used to define the head section of an HTML document. It typically contains information about the document, such as the title, meta tags, and links to external resources.

3.body: This tag is used to define the body section of an HTML document. It contains the main content of the web page.

4.div: This tag is used to group together HTML elements and apply styles to them.

5.p: This tag is used to define paragraphs of text.

6. h1 to h6: These tags are used to define headings of different levels.

7.a: This tag is used to create links to other web pages or resources.

8.img: This tag is used to embed images in an HTML document.

9. ul: This tag is used to create an unordered list.

10.ol: This tag is used to create an ordered list.

11.li: This tag is used to define list items in an unordered or ordered list.

12.table : This tag is used to create a table.

13.tr : This tag is used to define a row in a table.

14. td : This tag is used to define a cell in a table.

HTML Attributes

HTML attributes provide additional information about an element. They are always specified in the start tag and come in name-value pairs.



HTML attributes are used to provide additional information about an HTML element. They are used to modify or extend the default behavior of an element. Attributes are added to an HTML element using the opening tag of the element.
Here are some common HTML attributes:

1. id: Used to specify a unique identifier for an element. This identifier can be used to manipulate the element using JavaScript or to create links to a specific section of a web page.

2. class: Used to specify one or more classes for an element. This can be used to apply CSS styles to specific elements or to group elements for JavaScript manipulation.

3. style: Used to specify inline styles for an element. Inline styles override any styles defined in external or internal stylesheets.

4. href: Used to specify the URL for a link. This is used in the a element to create clickable links to other web pages, sections within the same page, or downloadable files.

5. src: Used to specify the URL for an image, audio or video file, or other external resource. This is used in the img, audio, video, and source elements, among others.

6. alt: Used to specify alternative text for an image, which is displayed if the image cannot be loaded or for accessibility purposes.

7. title: Used to provide additional information about an element, typically displayed as a tooltip when the user hovers over the element.

HTML Links

HTML links are used to connect one web page to another. They are created using the <a> tag.
